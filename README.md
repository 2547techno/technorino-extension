## Install

- Install dependencies (just cli tools): 

    ```bash
    npm install
    ```

## Just build

- Just build (chrome/v3 manifest):
    ```bash
    npm run build:v3
    ```

- Just build (firefox/v2 manifest):
    ```bash
    npm run build:v2
    ```

## Build & Run

- Build & run (chrome, v3 manifest):
    ```bash
    npm run chrome
    ```

- Build & run (firefox, v2 manifest):
    ```bash
    npm run firefox
    ```
